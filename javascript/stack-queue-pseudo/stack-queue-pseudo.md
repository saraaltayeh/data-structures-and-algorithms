# stack-queue-pseudo

## Methods

- enqueue
Arguments: value
Inserts value into the PseudoQueue, using a first-in, first-out approach.
- dequeue
Arguments: none
Extracts a value from the PseudoQueue, using a first-in, first-out approach.
